🚀 Quick Start

1

Install

# Clone & install
git clone repo-url
cd web-scraper-to-sheets
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
2

Run

# Basic scrape (5 pages)
python main.py

# Save to CSV
python main.py -o data.csv

# Custom pages
python main.py --pages 10
3

Export

# Export to Google Sheets
python main.py --export

# Full pipeline
python main.py --pages 10 \
  --export -o data.csv \
  --verbose

🧩 Features by Stage

🔄

Retry Mechanism

# 3 attempts with exponential backoff
Attempt 1: failed → wait 2s
Attempt 2: failed → wait 4s
Attempt 3: success!
  • ✓ 3 retries with 2x backoff (2s, 4s, 6s)
  • ✓ Handles timeouts, 5xx errors, connection issues
  • ✓ Skips 404 pages (no retry)
  • ✓ 15-second request timeout
🎭

User-Agent Rotation

# 5 different browser identities
Chrome 120 (Windows)
Firefox 121 (Windows)
Safari 17.2 (macOS)
Edge 120 (Windows)
Chrome 120 (macOS)
  • ✓ Random selection per request
  • ✓ Realistic browser fingerprints
  • ✓ Reduces detection risk
  • ✓ Easy to add more UA strings
📄

Pagination

# Auto-navigates through pages
/page/1/ → /page/2/ → ... → /page/N/
# Stops if page returns 404 or no data
  • ✓ Configurable page count (1-100)
  • ✓ Polite 1-3s delay between pages
  • ✓ Auto-stops on empty pages
  • ✓ URL pattern: /page/{n}/
🕷️

HTML Parsing

# BeautifulSoup4 extraction
soup.find("div", class_="quote")
  → title, description, price,
    date, link, tags
  • ✓ BeautifulSoup4 + html.parser
  • ✓ Structured dict output per item
  • ✓ Error-safe (skips broken HTML)
  • ✓ Easy to customize for other sites

Text Cleaning

# Input (raw):
"\u201cHello   World\u201d\n  "

# Output (clean):
"Hello World"
  • ✓ Removes Unicode quotes (\u201c \u201d \u2018 \u2019)
  • ✓ Strips whitespace & control characters
  • ✓ Collapses multiple spaces
  • ✓ Handles em-dash \u2014 conversion

Price Validation

# Handles all formats:
validate_price("$12.99")  → 12.99
validate_price("12,50")   → 12.50
validate_price("free")    → 0.00
validate_price(None)      → 0.00
  • ✓ Extracts numbers from strings
  • ✓ Handles European comma format
  • ✓ Defaults invalid prices to 0.0
  • ✓ Rounds to 2 decimal places
🔍

Deduplication

# Removes duplicates by key:
items = [
  {link: "/a", title: "First"},
  {link: "/b", title: "Second"},
  {link: "/a", title: "Dupe!"},  ← removed
]
  • ✓ Configurable dedup key (default: link)
  • ✓ Keeps first occurrence
  • ✓ Logs number of duplicates found
  • ✓ O(n) performance with set lookup
